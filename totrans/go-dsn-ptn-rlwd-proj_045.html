Built-in identifiers

Go comes with a number of built-in identifiers. They fall into different categories, including types, values, and built-in function.

Types

The following identifiers are used for Go's built-in types:

Category

Identifier

Numeric

byte, int, int8, int16, int32, int64, rune, uint, uint8, uint16, uint32, uint64, float32, float64, complex64, complex128, uintptr

String

string

Boolean

bool

Error

error

Values

These identifiers have preassigned values:

Category

Identifier

Boolean constants

true, false

Constant counter

iota

Uninitialized value

nil

Functions

The following functions are available as part of Go's built-in pre-declared identifiers:

Category

Identifier

Initialization

make(), new()

Collections

append(), cap(), copy(), delete()

Complex numbers

complex(), imag(), real()

Error Handling

panic(), recover()
